# FortiEscrow Scripts

Utility scripts for deployment, testing, and automation.

## Structure

- **`deployment/`** - Compile and deploy automation
- **`testing/`** - Test automation and coverage
- **`utils/`** - Utility scripts

## Quick Commands

```bash
# Compile
./scripts/deployment/compile.sh

# Deploy to testnet
./scripts/deployment/deploy_testnet.sh

# Run all tests
./scripts/testing/run_all_tests.sh

# Security tests only
./scripts/testing/run_security_tests.sh

# Coverage report
./scripts/testing/coverage_report.sh
```
