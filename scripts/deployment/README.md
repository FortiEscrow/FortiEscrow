# Deployment Scripts

Compile and deploy automation.

## Scripts

- `compile.sh` - Compile contracts to .tz
- `deploy_testnet.sh` - Deploy to Ghostnet
- `deploy_mainnet.sh` - Deploy to mainnet

## Usage

```bash
# Compile
./scripts/deployment/compile.sh

# Deploy to testnet
./scripts/deployment/deploy_testnet.sh

# Deploy to mainnet (requires careful review)
./scripts/deployment/deploy_mainnet.sh
```
