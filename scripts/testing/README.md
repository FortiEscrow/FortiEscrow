# Testing Scripts

Test automation and coverage.

## Scripts

- `run_all_tests.sh` - Run complete test suite
- `run_security_tests.sh` - Security tests only
- `coverage_report.sh` - Generate coverage report

## Usage

```bash
# Run all tests
./scripts/testing/run_all_tests.sh

# Security tests only
./scripts/testing/run_security_tests.sh

# Coverage report
./scripts/testing/coverage_report.sh
```
